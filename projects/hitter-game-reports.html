Hitter Game Reports

This project focused on creating hitter game reports to better visualize individual performances. The goal was to provide a tool that highlights how elite hitting performances. Using R to scrape game data, I built a script that filters, cleans, and analyzes the necessary information, then generates a series of plots to capture key aspects of a hitter's performance. The workflow began with a parameter-based filtering system to select hitters and isolate their performance on a specific day. I also created a separate script to plot outline of the stadium in which the game took place, the way I did it was by taking coordinates for every stadium and creating individial csv documents with each stadiums coordinates. The main script then pulled in the relevant stadium outline based on the game data, to create a spray chart that accurately reflects the context of the performance. From there, the script produced several plots:
  • Spray Chart – Displays each batted ball with color-coding by hit type (single, double, triple, home run) and scaled by distance, providing a clear view of a hitter’s power distribution and contact tendencies.
  • Strike Zone Plot – A visualization of the strike zone that shows where the pitches a batter put into play were located. Each pitch is labeled with its type and the resulting batted-ball outcome, helping identify which zones and pitch types a hitter handles most effectively.
  • Hit Summary Table – Provides detailed context on each batted ball, including pitch type, result, exit velocity, launch angle, and distance.
  • Hitting line table – provides traditional box score context alongside the advanced visuals.
After generating these plots, I used a complementary Python script to assemble them into a single 1080 x 1080 PNG image, making the reports easy to share on social media. Overall, this system generates concise, visually rich reports that make it easy to quickly understand a hitter’s performance and results, while also providing the depth needed for more detailed evaluation..
